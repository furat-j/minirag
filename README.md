# mini-rag

A minimal Retrieval-Augmented Generation (RAG) backend built with FastAPI for question answering.

This project is intended for learning and experimentation with RAG concepts.

---

## Tech Stack

- Python 3.8+
- FastAPI
- Uvicorn
- PyCharm (recommended)

---

## Setup (PyCharm Only)

This project is designed to be run entirely using PyCharm.

### 1. Create the project

1. Open PyCharm
2. Click **New Project**
3. Select **FastAPI** (or Python if FastAPI template is unavailable)
4. Choose **New Virtualenv**
5. Select Python 3.8 or later
6. Create the project

PyCharm will automatically configure the virtual environment and interpreter.
